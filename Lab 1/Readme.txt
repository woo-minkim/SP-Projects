프로젝트 개요 설명 
프로젝트 제목: MyLib 자료구조 분석 및 구현
프로젝트 설명: MyLib 프로젝트는 커널 프로그래밍에서 자주 사용되는 리스트, 해시 테이블, 비트맵 자료구조를 분석하고 구현하는 작업을 포함합니다. 이 프로젝트는 자료구조의 효율적인 처리를 위한 라이브러리를 생성하고, 해당 기능을 시연하는 인터랙티브 프로그램을 개발하는 것을 목표로 합니다.
주요 목표:
1.	자료구조 분석 및 구현:
o	리스트 (List):
	이중 연결 리스트(doubly linked list)를 분석하고 구현했습니다.
	struct list_elem과 struct list_item을 설계 및 구현하여 리스트 요소와 데이터를 분리하여 저장했습니다.
	리스트 초기화 함수(list_init), 리스트 시작과 끝 요소 반환 함수(list_begin, list_end), 리스트 삽입/삭제 함수 등을 구현했습니다.
o	해시 테이블 (Hash Table):
	키와 값을 연관시키는 해시 테이블을 구현했습니다.
	특정 해시 함수를 사용하여 키를 해시 값으로 변환하고, 해당 값을 기반으로 적절한 버킷에 저장했습니다.
	리스트 구조를 사용하여 해시 충돌을 해결했습니다.
	해시 테이블 초기화 함수(hash_init), 해시 테이블 삽입/삭제 함수 등을 구현했습니다.
o	비트맵 (Bitmap):
	각 비트를 Boolean 값으로 저장하는 비트맵 자료구조를 구현했습니다.
	메모리 공간의 낭비를 줄이고 효율적인 비트 연산을 지원하도록 설계했습니다.
	비트맵 생성 함수(bitmap_create), 비트 설정 함수(bitmap_set), 비트 카운트 함수(bitmap_count) 등을 구현했습니다.
2.	인터랙티브 프로그램 개발:
o	구현된 자료구조와 상호작용할 수 있는 명령어 기반 인터페이스 프로그램을 개발했습니다.
o	사용자가 명령어를 입력하고 자료구조의 기능을 확인할 수 있도록 했습니다.
o	예시 명령어:
	create list <LIST>: 리스트 생성
	create hashtable <HASH TABLE>: 해시 테이블 생성
	create bitmap <BITMAP> <BIT CNT>: 비트맵 생성
	delete <LIST | HASH TABLE | BITMAP>: 자료구조 삭제
	dumpdata <LIST | HASH TABLE | BITMAP>: 자료구조의 데이터 출력
	quit: 프로그램 종료
구체적인 구현 사항:
•	리스트 (List):
o	리스트 항목의 삽입, 삭제, 탐색 기능을 제공하는 이중 연결 리스트 구현
o	struct list_elem과 struct list_item 설계 및 구현
o	list_init, list_begin, list_end, list_push_front, list_push_back, list_remove 등의 함수 구현
•	해시 테이블 (Hash Table):
o	키와 값을 연관시키는 해시 테이블 구현
o	해시 함수(hash_int)와 해시 테이블 초기화, 삽입, 삭제 함수 구현
o	해시 충돌 해결을 위한 리스트 구조 사용
•	비트맵 (Bitmap):
o	Boolean 값을 저장하는 비트맵 자료구조 구현
o	비트맵 생성, 비트 설정, 비트 카운트 함수 구현
o	효율적인 메모리 사용을 위한 설계
프로젝트 결과: 이 프로젝트를 통해 커널 프로그래밍에서 자주 사용되는 자료구조의 구현 및 최적화 방법을 깊이 있게 이해할 수 있었으며, 명령어 기반 인터페이스 프로그램을 통해 실습 및 테스트가 가능하도록 하였습니다.
